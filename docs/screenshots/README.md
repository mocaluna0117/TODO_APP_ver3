# スクリーンショット置き場

ポートフォリオ資料（`docs/PORTFOLIO.md`）から参照する画面キャプチャをここに置きます。

## 撮り方

iOS シミュレータで起動し、`⌘ + S` でデスクトップに保存するのが手軽です。

```bash
cd todo_app
flutter run
```

## 必要なファイル名

`docs/PORTFOLIO.md` は以下のファイル名で画像を参照しています。撮影した画像を同じ名前で置いてください。

| ファイル名 | 撮る画面 |
|---|---|
| `home.png` | 「やること」タブ |
| `today.png` | 「今日やること」タブ |
| `done.png` | 「完了済み」タブ |
| `future.png` | 「やりたいこと」タブ |
| `add_task.png` | タスク追加 / 編集ダイアログ |
| `settings_1.png` | 設定画面（アプリ名・タブ・タグ） |
| `settings_2.png` | 設定画面（動作設定・並び順・カラーテーマ） |
| `settings_3.png` | 設定画面（カラーテーマ・バックアップ・データ管理） |

> ファイル名を変えたい場合は、`PORTFOLIO.md` 内の画像パスも合わせて変更してください。
